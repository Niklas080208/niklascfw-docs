# Willkommen bei den NiklasCFW Docs

Diese Dokumentation führt dich mit einem **roten Faden** von der **Vorbereitung & Installation** über **Backups** bis zu **Nachbereitung** und **Vertiefung** rund um die Nintendo Switch. Einstieg für alle Schritte: **[Start: Roter Faden](/switch/vorbereitung/roter_faden)**. **OmniNX ist der primäre Installationspfad**, der NiklasCFW Pack wird als Legacy-Fallback geführt. Alle Seiten findest du in der **Seitenleiste** unter „Nintendo Switch“.
